# 基于深度学习的生活垃圾检测系统

本项目是一个图形化的生活垃圾检测桌面应用，基于 **YOLOv8m** 目标检测算法、**PyQt5** 界面框架和 **FunASR** 离线语音识别技术，支持实时摄像头检测、静态图片检测和视频文件检测，并提供语音指令控制、检测结果语音播报、检测记录存储与导出等完整功能。

## ✨ 主要功能

- **三种检测模式**：实时摄像头、单张图片、视频文件
- **高精度识别**：基于 YOLOv8m 的生活垃圾检测模型，支持 43 个细分类别，4 个大类（可回收、厨余、有害、其他）
- **语音交互**：离线语音指令控制（开始/停止/暂停/切换模式），检测结果自动语音播报
- **数据管理**：检测记录自动存入 SQLite 数据库，支持按时间筛选查询
- **结果导出**：支持导出为 Excel、CSV、Word 报告
- **参数可调**：实时调整置信度阈值
- **界面友好**：左右分栏布局，检测画面、结果列表、统计图表一目了然

## 🏗 技术栈

| 模块         | 技术                                        |
| ------------ | ------------------------------------------- |
| 深度学习框架 | PyTorch 2.0+、Ultralytics YOLO               |
| 目标检测模型 | YOLOv8m（训练 43 类生活垃圾）                  |
| 图形界面     | PyQt5                                        |
| 语音识别     | FunASR（Paraformer-zh 离线模型）              |
| 语音合成     | pyttsx3（SAPI 引擎）                          |
| 数据库       | SQLite                                       |
| 数据处理     | OpenCV、NumPy、Pandas                         |
| 导出报告     | openpyxl（Excel）、python-docx（Word）         |
